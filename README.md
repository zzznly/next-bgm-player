# next-bgm-player
이 프로젝트는 기존에 React.js로 개발했던 [mytones-bgm-player](https://github.com/zzznly/mytones-bgm-player) 를 Next.js로 마이그레이션한 프로젝트입니다.

<br/>

## Dependencies

- React@19
- Next@15
- TypeScript@5
- CSS Modules
- React-Query@5
- Zustand@5

<br/>

## Install & Run

```bash
npm install
npm run dev
```

로컬에서 [http://localhost:3000](http://localhost:3000) 을 열어 결과물을 확인하실 수 있습니다.
